# Ar-MTO

Ar-MTO is a DetaNet-based Molecular Tensor Orbital project.

The central research question:

> Can local equivariant tensor fields be assembled, under symmetry constraints, into stable, transferable, and chemically meaningful molecule-level response modes?

## Data

Data are stored separately on the HPC server (`/data/home/scwc008/run/xxy/MTO/data`) and are **not** tracked by Git.

## Repository Contents

This repository hosts code, configs, scripts, tests, and experiment documentation for the MTO-Net system.
